Python

Python é uma linguagem de programação de alto nível, amplamente utilizada por sua simplicidade e legibilidade. Criada por Guido van Rossum e lançada em 1991, ela se destaca pela sintaxe clara e concisa, o que a torna uma excelente escolha para iniciantes, mas também poderosa o suficiente para desenvolvedores experientes.

Powerpoint

O Microsoft PowerPoint é um software de apresentação desenvolvido pela Microsoft e parte do pacote Office. Lançado em 1987, o PowerPoint permite aos usuários criar slides para apresentações visuais, com suporte para texto, imagens, vídeos, gráficos, animações e transições. É amplamente utilizado em ambientes empresariais, educacionais e profissionais para transmitir informações de maneira clara e envolvente.

Php

PHP (Hypertext Preprocessor) é uma linguagem de programação de código aberto amplamente utilizada para o desenvolvimento de sites e aplicações web dinâmicas. Criada por Rasmus Lerdorf em 1994, originalmente como uma ferramenta para monitoramento de visitas em seu site, ela evoluiu para uma das linguagens mais populares para back-end de aplicações web.

Word

Microsoft Word é um processador de texto desenvolvido pela Microsoft e é parte do pacote Microsoft Office. Lançado em 1983, o Word se tornou um dos programas mais populares e amplamente utilizados para criação, edição e formatação de documentos de texto.

JavaScript

JavaScript é uma linguagem de programação de alto nível, dinâmica e interpretada, amplamente utilizada para criar interatividade e dinamismo em páginas web. Desenvolvida por Brendan Eich em 1995, JavaScript é uma das linguagens mais importantes no desenvolvimento web, sendo executada no lado do cliente (navegador), mas também podendo ser utilizada no lado do servidor (com tecnologias como Node.js).

Css3

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML ou XML. Criado para separar a estrutura do conteúdo da aparência visual, o CSS permite controlar o layout, as cores, fontes, margens, espaçamento e outros aspectos estéticos de uma página web.

Html5

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar e estruturar conteúdo na web. Desenvolvido por Tim Berners-Lee no início dos anos 1990, o HTML define a estrutura de páginas web, utilizando uma série de "tags" (ou elementos) para organizar texto, imagens, links, vídeos e outros tipos de conteúdo.

Excel

Microsoft Excel é um software de planilha eletrônica desenvolvido pela Microsoft, amplamente utilizado para análise, organização e visualização de dados. Parte do pacote Microsoft Office, o Excel foi lançado inicialmente em 1985 e desde então se tornou uma das ferramentas mais populares para tarefas como cálculos, criação de gráficos e gerenciamento de informações.

Git

Git é um sistema de controle de versão distribuído, utilizado para gerenciar e rastrear mudanças em projetos de software. Criado por Linus Torvalds em 2005, Git permite que múltiplos desenvolvedores trabalhem simultaneamente em um projeto sem sobrescrever as alterações uns dos outros, facilitando a colaboração e o gerenciamento de código.

C++

C++ é uma linguagem de programação de alto desempenho e propósito geral, criada por Bjarne Stroustrup no início dos anos 1980. Ela é uma extensão da linguagem C, adicionando recursos de programação orientada a objetos, como classes e herança, o que a torna mais poderosa e flexível. C++ é amplamente utilizada no desenvolvimento de sistemas que exigem alta performance, como jogos, software de sistemas, drivers, aplicativos em tempo real e software embarcado.
